# TypeRead Chrome Extension

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

TypeRead is a Chrome extension that lets you practice typing by rewriting any paragraph on any website. Improve retention and typing skills through interactive engagement with real web content.

## 🚀 Features
- **Interactive Typing Mode:** Activate typing mode directly from the popup window.
- **Smart Selection:** Hover and select any paragraph on a web page (includes a visual outline on hover).
- **Immersive Overlay:** Features a typing overlay with faded text, a blinking cursor, and real-time feedback.
- **Real-Time Corrections:** Incorrect characters are highlighted in red; press **Tab** to skip a word, or **Esc** to exit.
- **Performance Tracking:** Displays a stats summary panel (WPM, Accuracy, Time, Mistakes) after each session.
- **Clean Design:** Minimalist, high-contrast black & white UI with a blue accent.
