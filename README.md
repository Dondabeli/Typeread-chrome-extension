# TypeRead Chrome Extension

TypeRead is a Chrome extension that lets you practice typing by rewriting any paragraph on any website. Improve retention and typing skills through interactive engagement with real web content.

## Features
- Activate typing mode from the popup window
- Hover and select any paragraph on a web page (visual outline on hover)
- Typing overlay with faded text, blinking cursor, and real-time feedback
- Incorrect characters highlighted in red; skip word with Tab, exit with Esc
- Stats summary panel (WPM, Accuracy, Time, Mistakes) after each session
- Minimalist, high-contrast black & white UI with blue accent

## How to Install
1. Download or clone this repository to your computer.
2. Open Chrome and go to `chrome://extensions`.
3. Enable **Developer mode** (top right).
4. Click **Load unpacked** and select the `TypeReadExtension` folder.
5. The TypeRead icon will appear in your Chrome toolbar.

## How to Use
1. Click the TypeRead extension icon to open the popup.
2. Click **Start Typing Mode**.
3. Hover over any paragraph on the page (e.g., Wikipedia) and click to select it.
4. The typing overlay will appear. Start typing the paragraph!
   - Correct characters are revealed in black
   - Incorrect characters are highlighted in red
   - Press **Tab** to skip a word
   - Press **Esc** to exit typing mode
5. When you finish, a stats summary will appear in the bottom right.

## Permissions
- `activeTab`, `scripting` (to inject content script and overlays)
- Works on all websites with readable content

## No data is collected or stored.
